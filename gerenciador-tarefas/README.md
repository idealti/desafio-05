# 🚀 Gerenciador de Tarefas

Este é um **gerenciador de tarefas** desenvolvido em **Angular**, criado como parte do desafio proposto pela Ide@l. O objetivo é oferecer uma ferramenta simples e eficiente para que o usuário possa organizar e acompanhar suas atividades diárias.

## ✨ Funcionalidades

Ao acessar a aplicação, você tem a opção de:

-   **Criar uma nova tarefa:** Adicione novas tarefas de forma rápida e intuitiva.
-   **Visualizar tarefas cadastradas:** Acesse a lista completa de todas as tarefas salvas.

Para facilitar a organização, a seção de visualização de tarefas possui um **filtro** que permite navegar entre os diferentes status:

-   ✅ **Concluídas:** Mostra apenas as tarefas que já foram finalizadas.
-   ⏳ **Pendentes:** Exibe somente as tarefas que ainda precisam ser concluídas.
-   📋 **Todas:** Lista todas as tarefas, independentemente do status.
