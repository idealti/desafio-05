### Desafio

Seu objetivo é desenvolver um gerenciador de tarefas (Todo App) que permita criar, gerenciar e filtrar tarefas de forma prática.

#### Casos de uso

1) Adicionar tarefas com um campo de texto e um botão de adicionar impedindo a criação de tarefas vazias;

2) Listar todas as tarefas exibindo o status de cada uma (pendente ou concluída). Deverá conter botões para marcar/desmarcar, editar tarefa e excluir.

3) Deve ser possível filtrar as tarefas com as seguintes opções: Todas, Pendentes e Concluídas;

4) Deverá conter uma forma de ordenar a lista de tarefas com: Criadas recentemente e antigamente (pode incluir outras ordenações se achar necessário);

5) Salvar as tarefas utilizando um store global (localStorage ou uma solução equivalente) para que ao recarregar a tela seja possível visualizar as tarefas cadastradas;

#### Requisitos

- O projeto deve ser entregue com Vue.js 3 utilizando Options API;
- Utilize o Bootstrap 5 como framework css;
